Commerce POS loyalty points support
-----------------------------------
This module gives you a field widget that you can use with customer field on
POS order type entity here :
  /admin/commerce/config/order-types/pos/edit/form-display


USAGE
-----
- Enable the customer field if its disabled already and change its widget to
'Autocomplete (show loyalty points)'
- Now head back to /admin/commerce/pos and you can find 'Loyalty points' widget.
- Select a custmer and hit 'Check points' button to fetch/see loyalty points of
  this customer.
